# MlbHeadlines

This Ruby Gem provides a CLI to view the Latest MLB Player Information, injuries, trades, etc.



## Installation

$ gem install mlb_headlines

## Usage

Type the below and follow the on screen prompts.
$ bundle install

$ bin/console
